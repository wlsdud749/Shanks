# Shanks
팀프로젝트
상크스 (상권정보 ! 크고 ! 스마트하게 !)


2023. SW 전문인재 양성과정에서 진행한 팀 프로젝트

사용 기술

- JAVA 11
- Spring
- Spring Data JPA
- Jquery
- Ajax
- MySQL
- HTML, CSS, JAVASCRIPT
사용 툴

- INTELIJ
- VSCODE
- TALEND

기능

- 사용한 데이터

공공데이터의 상권정보를 이용
(그 중 동구의 상권만 이용하였다)

로그인 및 회원가입 - 박동진

지도 - 손진영, 배현모, 하성재

-기능
1. 로그인 및 회원가입
2. 게시판
3. 카카오 지도를 활용한 지도정보 (중심좌표는 동구)
4. 가게 검색
5. 가게 위치에 따른 마커 생성
6. 마커를 클릭하면, 해당 카페의 정보 출력

-보완해야 할 점
1. css
2. 관심목록
3. 가게의 정보 뿐만 아니라 다른 상권 정보
